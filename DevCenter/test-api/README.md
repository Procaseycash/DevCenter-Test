# Nest framework TypeScript starter

Starter repository for [Nest](https://github.com/kamilmysliwiec/nest) framework.

### Installation

```
$ npm install
```

### Serve

```
$ npm run serve
```
### Application runs at port 3000

http://localhost:3000/
